Prueguntas a Responder en el ejercicio 1

A continuacion paso a enumerar las preguntas que debemos responder en el ejercicio y luego seran resueltas punto por punto

Preguntas:

  • Dar un par de ejemplos de uso de etiquetas Meta
  • Nombrar al menos 2 atributos aplicables a la etiqueta HTML
  • ¿Como se relaciona un header http con la etiqueta head de HTML?
  • ¿Que es UTF8?
  • ¿Que atributo HTML sirve para aplicar estilo, a que etiqueta es aplicable?
  • ¿Donde se indica que un enlace debe ubicar el documento en una nueva pestaña?
  • Explicar cuando usarían etiquetas semánticas y con que propósito.
  • ¿Que atributos de estilo básicos aplicarían a una tabla? Explicar.
  • ¿Que diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?
  • ¿Que atributos marcan la agrupación de celdas de una tabla?. ¿A que etiquetas se aplican?

Respuestas:

  • - Meta charset: Sirve para codificar los caracteres y evitar errores si la fuente no fue previamente definida en el header del protocolo HTTP.

    - Meta description: Aqui podemos describir brevemente el contenido de la página web. Esta información se muestra como snippet (una síntesis en dos líneas del tema de una página que aparece bajo la URL) en los buscadores, por lo que se recomienda cuidar su redacción.

  • - Lang: Indica en lenguaje del texto

    - Id: Codigo unico identificador de la etiqueta

    - Class: Establece la clase Css que se aplica a los elementos

  • El contenido de Head provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos. Estos metadatos son precisamente los que viajan en el header del request del protocolo Html para que el servidor pueda devolver el documento con todos sus componente y su formatos correcto.

  • Utf 8 es un formato de codificacion que permite codificar miles de caracteres de Unicode para que sean entendibles por la maquina.

  • El atributo que sirve para aplicar estilo es "Style". y se puede aplicar en cualquier etiqueta del body.

  • Se debe indicar dentro de la etiqueta "a", mas precisamente en el atributo "target" con el valor "_blank".

  • Las etiquetas semanticas se utilizan cuando necesitamos marcar los limites de distintos tipos de contenido. Su proposito es el de delimitar el contenido del documento, sin tener nada que ver con la estructura sintactica del mismo. Delimitar el contenido puede ayudar a ser mas organizado a la hora de codificar la pagina web y ademas permite aplicar estilos de manera mas ordenada.

  • - Border: Establecer un borde visible y de algun color

    - Padding: Esto lo haria para dar espacio entre cada columna y fila

    Text-align: Y esto seria para alinear todos los datos en la tabla.


  • El elemento body de HTML representa el contenido de un documento HTML. En cambio el body del protocolo HTTP representa las variables u archivos que los requerimientos o las repuestas envian el uno al otro. De hecho una respuesta a una peticion para ingresar a una web es la url de esa web, que cuando se ejecuta el navegador lee el HTML y imprime en pantalla su body.

  • Los atributos que marcan la agrupacion de celdas en un tabla son colspan y rowspan y se aplican a la etiqueta td(celda con datos) y th(celda con titulo).
